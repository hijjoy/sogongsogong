### 📍 세팅 방법

1. pip list를 설치

```
$ pip install -r requirements.txt
```

2. 장고 서버 실행

```
$ python manage.py runserver
```

3. my-react-app 에서 리액트를 실행

```
$ cd my-react-app
```

```
$ npm install axios
```

```
$ npm install react-router-dom
```

```
$ npm run start
```

4. 잘 동작하는지 확인 🎉

```
http://127.0.0.1:3000/login
```

#### ✓ 로그인 후 새로고침을 한번 해야 실행 됩니다 🥺
